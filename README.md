# JExcelDownloader - Excel Download Module

## jitpack (with maven)
```
    <repositories>
        <repository>
            <id>jitpack.io</id>
            <url>https://jitpack.io</url>
        </repository>
    </repositories>
    
    <dependency>
        <groupId>com.github.gamzagamza</groupId>
        <artifactId>JExcelDownloader</artifactId>
        <version>{version}</version>
    </dependency>
```
